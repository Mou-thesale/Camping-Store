📦 Getting Started

1. Local Deployment

To view the store locally on your machine:

Download or clone this repository.

Open index.html in any modern web browser.

2. Live Hosting (GitHub Pages)

This site is optimized for GitHub Pages. To make it live:

Go to Settings > Pages in this repository.

Select the main branch and click Save.

Your site will be live at: https://mou-thesale.github.io/Camping-Store/

🔐 Administration & Management

The Admin Vault is located at the very bottom of the website. Once authenticated with your secure password, you can:

Manage Gear: Add new products or remove out-of-stock items instantly.

Publish Stories: Write and edit blog posts to share camping tips and field tests directly with your community.
